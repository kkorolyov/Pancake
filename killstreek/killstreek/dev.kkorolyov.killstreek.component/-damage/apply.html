fun apply(health: Health, dt: Long): Boolean

Applies the negation of this damage's value to some health. If this damage has a duration, the appropriate portion of its value is applied instead. If this damage has expired, this method does nothing.

Return

{@code false} if expired, else {@code true}

Parameters

health

damaged health

dt

{@code ns} elapsed since previous invocation of this method
